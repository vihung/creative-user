The Game of Literature

Rules

Players and Teams

  • the game is played with six players.
  • players are split into two teams of three each

the Deck

  • all twos are discarded from the standard deck of cards leaving 48 cards, 3 to A of all suits
  • deck is conceptually split into eight sets - a lower set 3-8 (inclusive) and higher set 9-A (inclusive) of each suit

Game

  • deck is shuffled
  • players are dealt the cards - eight cards each
  • aim is for a team to accumulate as many sets as possible. higher set has 5 points, lower has 3

Turn Play

  • the player whoâ€™s turn it is asks a player from the opposite team for a named card, e.g. 8 of hearts
  • if the asked player has it, it must be given and turn stays with the asking player
  • asking player can only ask for a card in a set for which he has at least one card in hand, e.g 8 of hearts only if he has at least one card in the higher hearts set
  • if the asked player does not have the card, turn passes to him

Claiming a Set

  • if the player whose turn it is is convinced that their team holds an entire set between them, they can claim it
  • the claiming player must show their own cards for that set
  • the claiming player must declare which team member holds which card
  • if they are correct, the team gets the points for that set
  • if they are wrong, the opposing team claims the set and gets the points

End Game

  • the game ends when all sets have been claimed
  • The winning team is the one with the most points

Bluffing

  • it is allowed to ask for a card already in your hand
  • it is not allowed to ask for a card if you do not hold any cards in the same set
  • it you hold the card that is asked of you, you must provide it

Signalling

  • players in a team should play co-operatively with one another, against the opposing team
  • players are not allowed to talk or confer or in any direct way communicate the cards in their hand
  • when one player asks for a card, they are signalling that they are trying to accumulate that set
  • other team members should attempt to continue the call for cards in that set (if they can do so per the rules above)
  • if a player starts a call for a particular card and a team mate has that card they can signal this to their team mates by starting another call for a card in the same set when it is their turn
